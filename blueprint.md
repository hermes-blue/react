# Blueprint: 퍼스널 스타일리스트 서비스 (Personal Stylist)

## 서비스 목적 및 기능
사용자의 신체 정보(키, 몸무게)와 사진을 분석하여 개인 맞춤형 스타일 가이드를 제공하는 프리미엄 AI 스타일링 서비스입니다.

## 프로젝트 개요
- **기술 스택**: React 19, TypeScript, Vite
- **아이콘**: lucide-react
- **스타일링**: 고급스러운 그라데이션, 깊이감 있는 그림자, 인터랙티브 글로우 효과를 적용한 Modern CSS
- **현재 버전**: 사용자 기본 정보 입력 단계

## 현재 계획
1. **사용자 프로필 화면**:
    - 프로필 사진 업로드 (미리보기 지원)
    - 키 (cm) 및 몸무게 (kg) 입력 필드
2. **반응형 디자인**: 모바일 및 웹 환경 최적화
3. **고급 UI/UX**: GEMINI.md 지침에 따른 세련된 타이포그래피 및 시각 효과 적용
